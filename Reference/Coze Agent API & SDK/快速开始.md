- 流式响应 ：智能体在生成回复的同时，将回复消息以数据流的形式逐条发送给客户端。处理结束后，服务端会返回拼接后完整的智能体回复。详细说明可参考 ​ 流式响应 。 ​

- 非流式响应 ：无论对话是否处理完毕，立即发送响应消息。开发者可以通过接口 ​ 查看对话详情 。查看对话详情确认本次对话处理结束后，再调用 ​ 查看对话消息详情 接口查看模型回复等完整响应内容。详细说明可参考 ​ 非流式响应 。 ​
