# 移除企业成员

> 来源: https://www.coze.cn/open/docs/developer_guides/remove_enterprise_member

- 套餐限制 ：扣子企业版（企业标准版、企业旗舰版）。 ​

- 需要先调用 移除组织成员 API 将用户从所有组织中移除，再执行本 API 将其移出企业。 ​

- 被移除成员的资源只能转移给企业超级管理员或企业管理员。 ​

- 每次请求只能移除一位成员。如需移除多位成员，请依次发送请求。 ​

- 该 API 不支持并发请求。 ​

| 请求方式​ | DELETE​ |
| 请求地址​ | ​Plain Text复制https://api.coze.cn/v1/enterprises/:enterprise_id/members/:user_id​​ |
| 权限​ | Enterprise.removePeople​确保调用该接口使用的访问令牌开通了企业特权应用中的Enterprise.removePeople权限，详细信息参考OAuth JWT 授权（企业特权应用）。​ |
| 接口说明​ | 移除企业成员。​ |

| 参数​ | 取值​ | 说明​ |

| Content-Type​ | application/json​ | 解释请求正文的方式。​ |

| 参数​ | 类型​ | 是否必选​ | 示例​ | 说明​ |
