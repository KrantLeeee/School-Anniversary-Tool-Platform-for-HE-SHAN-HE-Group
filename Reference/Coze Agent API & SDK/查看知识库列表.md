# 查看知识库列表

> 来源: https://www.coze.cn/open/docs/developer_guides/list_dataset

- 暂不支持通过 API 查看低代码应用中的知识库。 ​

- 暂不支持通过该 API 查看火山知识库中的文件列表等详细信息。 ​
